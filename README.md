# Landmark-aligmenttest

Benchmark face-alignment pipelines on TinyFace for CVLFace ViT-KPRPE:

- DFA MobileNet
- DFA ResNet50
- SCRFD-10G-KPS
- MediaPipe FaceMesh
- Square-resize baseline

TinyFace data and downloaded model weights are intentionally excluded from this repository. See `CVLface-main/benchmarks/tinyface_alignment/README.md` for the Modal workflow.
